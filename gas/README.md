# Google Apps Script — Маршрутизация отчётов по группам

## Установка

1. **Добавьте колонку в лист Users**  
   В Google Таблице на листе `Users` добавьте новую колонку с заголовком **`group_chat_id`** (например, после колонки `Role`).

2. **Скопируйте код**  
   Содержимое `Code.gs` нужно вставить в ваш проект Google Apps Script (заменить им текущий скрипт).

3. **Безопасность**  
   Рекомендуется хранить `BOT_TOKEN` в свойстве скрипта, а не в коде:
   - Файл → Проект → Свойства → Script properties  
   - Добавьте ключ `BOT_TOKEN` со значением токена  
   - В коде: `const BOT_TOKEN = PropertiesService.getScriptProperties().getProperty('BOT_TOKEN');`

## Изменения в скрипте

- **getUserGroupChatId(userId)** — читает `group_chat_id` из листа Users по userId
- **getTargetChatId(userId)** — возвращает группу пользователя или `CHAT_ID` по умолчанию
- **sendReport** — отправляет отчёт в группу пользователя (если задана) или в группу по умолчанию
- **sendExcel** — то же самое
- **addUser / updateUser** — сохраняют поле `group_chat_id` при создании и редактировании пользователя
- **getUsers** — возвращает `group_chat_id` (через стандартную обработку всех колонок)

## Как получить ID группы

1. Добавьте бота в нужную группу.
2. Напишите в группе любое сообщение.
3. Откройте: `https://api.telegram.org/bot<ТОКЕН>/getUpdates`
4. Найдите `"chat":{"id": -1001234567890}` — это ID группы.
5. Укажите его в таблице Users или в админке приложения.
